Super Productivity Mods

Camada de personalização visual adicionada ao Super Productivity original. Cursor customizado, partículas, stamps arrastáveis e fundo interativo — tudo salvo localmente.

⇩ Baixar ZIP Ver código Projeto original

Funcionalidades

▶

Cursor customizado

Presets prontos (anime, coração, estrela, varinha, Mummy) ou suba seu próprio PNG/GIF para o estado normal e hover separadamente. Controle de tamanho e opacidade.

◆

Fundo personalizado

Cor sólida ou imagem (upload ou URL) com controles de desfoque e opacidade. Renderizado como camada fixa sem afetar o layout do app.

✦

Partículas

8 modos: estrelas, pétalas, neve, bolhas, confete, matrix, vagalumes e brilhos. Controles de quantidade, velocidade, opacidade e cor. Roda fora do Angular zone.

■

Stamps

Imagens arrastáveis sobre a interface. 4 formatos alteráveis on the fly: quadrado, oval, retângulo horizontal e vertical. Posição salva automaticamente.

Como instalar

1

Clone o repositório

git clone https://github.com/03s9/super-productivity.git
2

Instale as dependências

npm install
3

Rode no navegador

npx ng serve

Abra localhost:4200 no navegador. O botão de mods aparece no canto inferior direito.

4

Ou gere o executável

npx ng build --configuration development npx electron-builder --win --dir

O executável fica em .tmp/app-builds/win-unpacked/

Notas técnicas

Persistência

Toda configuração é salva em localStorage como base64. Funciona no Electron sem servidor.

Cursores .ani

O formato .ani (Windows Animated Cursor) não é suportado por nenhum browser moderno via CSS. Use PNG, GIF ou WebP.

Performance

Partículas e drag de stamps rodam fora do Angular zone para não impactar a detecção de mudanças do app.

Sem dependências novas

Nenhuma biblioteca adicional foi adicionada. Apenas APIs nativas: Canvas, Pointer Events, Web Audio.
