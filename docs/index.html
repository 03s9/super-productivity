Super Productivity Mods

Camada de personalizacao visual e recompensas adicionada ao Super Productivity original. Cursor customizado, partículas, stamps arrastaveis, fundo interativo e o novo Trinkets Addon — tudo salvo localmente.

Baixar ZIP Ver codigo Projeto original

Funcionalidades

C

Cursor customizado

Presets prontos (anime, coracao, estrela, varinha, Mummy) ou suba seu proprio PNG/GIF para o estado normal e hover separadamente. Controle de tamanho e opacidade.

F

Fundo personalizado

Cor solida ou imagem (upload ou URL) com controles de desfoque e opacidade. Renderizado como camada fixa sem afetar o layout do app.

P

Particulas

8 modos: estrelas, petalas, neve, bolhas, confete, matrix, vagalumes e brilhos. Controles de quantidade, velocidade, opacidade e cor. Roda fora do Angular zone.

S

Stamps

Imagens arrastaveis sobre a interface. 4 formatos alteraveis on the fly: quadrado, oval, retangulo horizontal e vertical. Posicao salva automaticamente.

T

Trinkets Addon

Sistema de recompensas com pets 3D colecionaveis. Complete tarefas para ganhar lootboxes, desbloquear companions e coloca-los na tela. Inclui economia de moedas, prateleira de trofeus 3D e ate 6 companions arrastaveis.

Como instalar

1

Clone o repositorio e mude para a branch mods

git clone https://github.com/03s9/super-productivity.git cd super-productivity && git checkout mods
2

Instale as dependencias

npm install
3

Rode no navegador

npx ng serve

Abra localhost:4200 no navegador. O botao de mods aparece no canto inferior direito.

4

Ou gere o executavel

npx ng build --configuration development npx electron-builder --win --dir

O executavel fica em .tmp/app-builds/win-unpacked/

Notas tecnicas

Persistencia

Toda configuracao e salva em localStorage como base64. Funciona no Electron sem servidor.

Cursores .ani

O formato .ani (Windows Animated Cursor) nao e suportado por nenhum browser moderno via CSS. Use PNG, GIF ou WebP.

Performance

Particulas e drag de stamps rodam fora do Angular zone para nao impactar a deteccao de mudancas do app.

Trinkets Addon

Usa Three.js para a prateleira 3D e os companions na tela. Adiciona as dependencias three e @types/three.
